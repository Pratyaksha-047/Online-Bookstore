Online Bookstore Guide

Home Page

Landing page of the website. By clicking on the three bars given at the top right corner, navigation slides in which has link to other pages. On home page, firstly there is intorduction section, then we have some feature. Following that top 5 most sold books are given. Below that we have available books section containing all the books, connected to individual pages. Lastly we have footer, which contains quick link to guide, footer, contact information and social media.

Available Books

Section given on the bottom of home page. It displays all the books that we sell and they are sorted genre wise. You can scroll them to see all the books. By clicking of the image of any book that you wish to buy, you will be redirscted to individual page of the book. To buy any book you have to login your account. If you are a new user then you will have to register first.

About Page

About page contains information regarding our bookstore. It displays all the feature that are present on our web application. Following feature section, we have sales section, which is currently static, but we plan to display all books on sale in this section in the coming future. After sales section we have offers section. It is not implemented yet but we consider it to be a future scope. Lastly we have our team, followed by footer.

Sales part

Sales section which is provided on the about page, it contains the information oof the books that are available on sale. It is a carousel of books, so you can move them left or right by clicking on the arrow given on left and right. Three books are visible at a time. This section is currently static, we plan to implement this feature in future.

Offers Available

Offers section present on the about page, contains all the offers that are provided to the user, such as 3 free trial for new user and 80% discount on first purchase, unlimited access to the audiobooks id the user subscribes to the bookstore, etc. This feature in not implemented yet howerever, we plan to make it work in the near future.

Search Available Book

Search book page contains all the books that are available for selling. User can search for the book of his/her choice by filling in the name of the book, or a word from the name of the book(e.g. Sherlock will render all books containing Sherlock as name in its title) or author name. If the book is not present then all the books will get rendered onto the page.

Contact Us Page

On contact page, contact information such as phone number, email id, timing and address are provided. By clicking on the email, user will be redirected to the mail app from where they can send mail to us. By clicking on the address, user is redirected to google map where location of our bookstore is given. User also has the option to send message throught the form given on the page.

Register Page

If you are a new user then you will have to register to our website in order to buy books which can be done only when you are logged into your account. To register, user will have to fill in his/her personal information such as name, phone number(including your country code), address, email id and password. If your are registered successfully then you will be directed to the login page.

Login Page

In order to login to your account, user has to fill in his/her email id along with password that he/she last set. If you are a new user and try to login, then an error will flash on the screen. If you are new user, you will have to register yourself first. If you have forgotten your password then you can choose forgot password option.

Forgot Password

If a user is registered but he/she has forgotten his/her password then, the person can choose forgot password option given on the login page. On the forgot password page, user can reset password, in order to do so, user has to enter email id and then a mail will be sent to the user from where then=y can change password.

Account Page

If the login is successful, then the option given in the navigation bar are updated. Now user will have access to the account page and he/she can buy books. On account page, User have the option to update his/her details and password. User can access the order history from the same page.

Change Account Details

In order to change account details, user has to click on edit account button given on the account page, then he/she can edit the details as per his/her requirement. Once the user make changes, then he/she can click on update button to save all the changes.

Change Password

In order to change password, user has to click on the reset passsword button given on the change password button given on the account page. User can the enter new password and by clicking on the 'Update password' button he/she can save the changes.

Order History

Order history of a user can be accessed by clicking on the 'oder history' div given on the left side of the account page. All the information associated to the payment is visible on the page, such as order id, book id, book name, book price and order date.

Individual Book

Individual book page can be accessed from the home page or the search page. Whichever book that the user wants to buy, he/she can visit this page. All the information related to the book is given on the page such as book title, author's name, price, description, genre, publishing year and the books available in the stock. For buying the book, user has to click on buy button given on the page.

Buy Book & Payment Option

In order to buy a book, user has to visit that book's page and then click on buy button, after clicking on the buy button, a payment popup will appear having all the payment option. User has to choose the option through which he/she will make payment. After successful payment, a popup will appear which confirms if the user want the receipt on his/her mail of not. After confirming user will be directed the order history page.

Terms & Condition Page

Terms and condition of our website are given on this page, user can access this page from the footer which is present on the home page and about page. Read all the terms and condition carefully and then click on agree button given on the bottom left corner.

Guide Page

Guide is the page that you are currently on, this page contains information regarding all the pages which are present on this website except the pages which are only accessed by the administrator of the online bookstore. By clicking on any of the image, user will be redirected to the webpage which is given in the image.

Footer

Footer contains quick link to guide, terms and condition page and social media. It also contains contact information such as phone number, timing, email id and location of the bookstore. By clicking on the email, user will be redirected to the mail app from where they can send mail to us. By clicking on the address, user is redirected to google map where location of our bookstore is given. It is present on the home page and about page.
